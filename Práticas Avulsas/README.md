## Projetos

### Gerador de Mensagens INSS

Aplicação desenvolvida em Python com Tkinter para automatizar a geração de mensagens para portabilidade de empréstimos consignados.

**Tecnologias:**
- Python
- Tkinter

**Funcionalidades:**
- Geração automática de mensagens
- Formatação monetária
- Copiar para área de transferência
